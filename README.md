Configuracion inicial del proyecto:
1.crear entorno virtual(en VSC) 
    python -m venv .venv
2.instalar Django
    pip install django
3.crear proyecto Django
    django-admin startproject nombre_proyecto
4.instalar Pillow
    pip install pillow
5.instalar Cryptography
    pip install cryptography
